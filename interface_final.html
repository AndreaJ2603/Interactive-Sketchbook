Interface System Design

For this interface system I have modified the navigation model to make it more consistent.

I have revised the structural model to implement Norman’s Principles and such as affordance signifiers and constraints.

Especially for the microproject I indicated the signifier by labeling a tap button to inform the user clearly that an action was needed to start playing the game. I also added a constraint so only one screen or keyboard action can happen per step.

Throughout the interface design I made sure to apply Nielsen’s Heuristics principle, especially the principle of minimalist design.

I aimed to keep the screen focused on only what users need to navigate the system effectively, to reduce visual clutter and improve usability.

Additionally, in every page the navigation shows flexibility and efficiency because I have implemented the going back/home button. This was major for me because it was the greatest mistake of the previous project that I wanted to correct.

Another heuristics that I have focused on was the user control and freedom by adding tabs on the microproject especially that would allow the user to start a new game any time during the interaction. Also informed users how to play the game to score points.

I think that it is interesting that the Normans and Shneiderman principle are close when it comes to these standards and rules. I incorporated Shneiderman eight golden rules by implementing consistency. The interface design displays the same actions and layouts and language in every page. I think having that in place already increases the user’s trust in the system.

The system also provides real-time feedback by displaying score and time as users progress through the game, helping reinforce engagement and awareness of performance.

Another golden rule that I think was well used on this design was the short term memory load. The system is easy to navigate and the user action has an expected response. For example, each menu option clearly corresponds to a distinct function, such as “Resume Game” or “New Game,” ensuring that each action leads to an appropriate outcome.

Overall, the interface design I have aimed to create a clear, consistent, and user-centered experience by incorporating and applying established usability principles to support the users navigation. My goal was also to create a navigation system with meaningful interactions.
